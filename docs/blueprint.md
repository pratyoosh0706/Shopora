# **App Name**: Shopora: AI-Powered Shopping

## Core Features:

- User Authentication: Secure sign-up and sign-in system with interest selection, storing user data in the cloud database.
- Admin Dashboard: Restricted access for admin to manage products and orders.
- Product Management: Functionality for admins to add, edit, and delete products.
- AI Assistant (Nova): AI-powered chatbot providing personalized product recommendations based on user interests and behavior; acts as a tool for discovery.
- Smart Target Marketing: AI dynamically updates product recommendations based on user interactions (searches, views, purchases).
- Product Display: Display 100 preloaded products including name, description, price (â‚¹ INR), category, and optional image; use data from cloud database.

## Style Guidelines:

- Primary color: Soft purple (#A096B3) to evoke trust and intelligence.
- Background color: Light gray (#F0F2F5) for a clean, minimalistic feel.
- Accent color: Teal (#45B8AC) for highlighting key elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font, providing a clean and modern appearance.
- Use simple, line-based icons for a clean and modern aesthetic.
- Minimalistic layout with ample white space to focus on products. Grid-based design for product listings.
- Subtle animations and transitions to enhance user experience.
